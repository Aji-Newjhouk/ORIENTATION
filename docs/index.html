A propos de notre site

Aji Newjhouk est une plateforme dédiée aux élèves de terminale et aux étudiants de bac+2, visant à les accompagner dans leurs choix académiques. Le site propose une multitude de ressources, notamment des guides d’orientation sur les différentes filières et options post-bac. Les utilisateurs peuvent bénéficier de tests d’intérêts interactifs pour mieux comprendre leurs compétences et aspirations.
